# basic-docker-nginx

### To run on local

```bash
# git clone this repository

$ cd basic-docker-nginx

# login to your Docker account
$ docker login

# pull nginx image for docker
$ docker pull nginx
 
# Run the server
$ docker-compose up 

# go to http://192.168.99.100:8080/ on windows or http://localhost:8080 on linux, mac
```
